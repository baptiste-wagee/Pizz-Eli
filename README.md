# 🍕 Pizz-Eli

**Pizz-Eli** est un projet personnel que j'ai développé afin de mettre en pratique mes compétences en **développement web** à travers la création d'un site moderne pour la pizzeria **Pizz'Eli**, située à **Sainghin-en-Weppes**.

Le site a été entièrement réalisé en **HTML**, **CSS** et **JavaScript**, avec pour objectif principal de progresser en développement front-end tout en proposant une interface moderne et agréable.

## 📖 À propos du projet

Pizz-Eli est un site web **one-page** présentant la pizzeria **Pizz'Eli** de Sainghin-en-Weppes.

Ce projet me permet de développer mes compétences en développement web, en design d'interfaces et en expérience utilisateur, tout en proposant une vision plus moderne de la présence en ligne d'une entreprise locale.

L'objectif n'est pas seulement de créer un site fonctionnel, mais également de développer mes compétences en :

- HTML5
- CSS3
- JavaScript
- Responsive Design
- Organisation d'un projet web
- UI / UX Design

## 🚀 Technologies utilisées

- 🌐 HTML
- 🎨 CSS
- ⚡ JavaScript

## 🎯 Objectifs

- Concevoir un site vitrine moderne pour une pizzeria
- Améliorer mes compétences en développement front-end ainsi qu'en conception d'interfaces (UI/UX)
- Découvrir de nouvelles techniques de mise en page, d'animations et d'interactions en JavaScript
- Réaliser un projet concret pouvant être enrichi avec de nouvelles fonctionnalités
